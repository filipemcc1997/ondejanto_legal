Política de Privacidade

Última atualização: junho de 2025

O OndeJanto ("nós", "a aplicação") é uma aplicação móvel que ajuda os utilizadores a decidir onde jantar, com base na sua localização e preferências. Esta política explica que dados recolhemos, como os utilizamos e quais são os teus direitos.

1. Dados que recolhemos

  • Localização (GPS) — usada exclusivamente para sugerir restaurantes próximos de ti. Não guardamos o historial de localização nos nossos servidores; as coordenadas são transmitidas apenas no momento do pedido de sugestão.
  • Conta (opcional) — se criares conta, guardamos o teu endereço de e-mail e nome fornecidos. A autenticação é gerida pelo Supabase.
  • Feedback e histórico — se estiveres autenticado, guardamos as tuas avaliações de restaurantes (gostei / não gostei) e o historial de sugestões para melhorar as recomendações futuras.
  • Restaurantes guardados — a tua lista de favoritos, associada à tua conta.
  • Dados de utilização anónimos — recolhemos eventos de uso agregados (ex.: quantas vezes foi pedida uma sugestão) através do PostHog, sem identificar o utilizador individualmente.

2. Como usamos os dados

  • Fornecer sugestões de restaurantes personalizadas ou aleatórias.
  • Melhorar a qualidade das recomendações (modo inteligente, desbloqueado após 5 avaliações).
  • Manter o historial e os favoritos sincronizados entre dispositivos (utilizadores autenticados).
  • Melhorar a aplicação com base em métricas de uso anónimas.

3. Partilha de dados

Não vendemos, alugamos nem partilhamos os teus dados pessoais com terceiros para fins comerciais. Os dados são processados pelos seguintes sub-processadores:

  • Supabase (autenticação e base de dados) — política de privacidade
  • Railway (alojamento da API) — política de privacidade
  • PostHog (analytics) — política de privacidade

4. Retenção de dados

Os dados de conta e historial são mantidos enquanto a conta estiver ativa. Podes solicitar a eliminação da tua conta a qualquer momento através das definições da aplicação ou por e-mail para o endereço indicado abaixo. Os dados de localização não são persistidos após o pedido de sugestão ser processado.

5. Os teus direitos (RGPD)

Se te encontras na União Europeia, tens direito a:

  • Aceder aos dados que guardamos sobre ti.
  • Corrigir dados incorretos.
  • Solicitar a eliminação dos teus dados ("direito ao esquecimento").
  • Retirar o consentimento para o tratamento de dados a qualquer momento.
  • Apresentar reclamação à Comissão Nacional de Proteção de Dados (CNPD).

6. Menores

O OndeJanto não se destina a crianças com menos de 13 anos. Não recolhemos intencionalmente dados de menores.

7. Alterações a esta política

Quaisquer alterações serão publicadas nesta página com a data de atualização revista. O uso continuado da aplicação após alterações constitui aceitação da política atualizada.
